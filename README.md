# Quantum Configuration

This is the syntax configuration for Quantum:

- Syntax highlighting
- Comment toggling `/*`
- Bracket matching `()`
- Bracket autoclosing `()` `''` `$$`
- Bracket autosurrounding `()` `$$`

## Recommended User Settings

```json
{    
    "files.defaultLanguage": "quantum",
    "files.eol": "\n",
    "files.insertFinalNewline": true
}
```
